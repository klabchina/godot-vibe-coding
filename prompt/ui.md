# Bow Survivor — UI 界面图生成提示词

## 游戏风格概述

- **游戏名**: Bow Survivor — Co-op Arrow Defense
- **美术风格**: 现代像素美术（类 Dead Cells），暗色调地下城氛围
- **色调基准**: 深蓝黑底色（#141420 ~ #0F0F1A），搭配暗金/青色/绿色高亮
- **分辨率**: 1980×1280（3:2 比例）
- **用途**: 以生成图作为 UI 界面的背景底图或设计参考稿，通过 Godot 程序叠加文本与按钮

---

## 生成参数（通用）

| 参数 | 值 |
|------|-----|
| 生成方式 | text2image |
| ratio | 3:2 |
| resolution_type | 2k |
| model_version | 5.0 |
| poll | 60 |

---

## CLI 模板

```bash
dreamina text2image \
  --prompt="[提示词]" \
  --ratio=3:2 \
  --resolution_type=2k \
  --model_version=5.0 \
  --poll=60
```

**界面结构**: 暗色全屏背景，画面正中央垂直排列：顶部 "BATTLE RESULT" 标题、大号等级字母（S/A/B/C，颜色各异）、5行统计数据（波次/击杀/伤害/经验/HP）、底部 "Return to Menu" 按钮。需支持 S 级金色、A 级青色、B 级绿色、C 级灰色等级展示的不同变体（共 4 张）。

## 使用说明

生成图作为全屏底图，通过 Godot 的 `TextureRect` 铺满整个 `Control`，锚点设为 `FullRect`。程序代码继续负责所有交互元素（标题文字、按钮、数据标签），只需在生成图上方叠加即可。
