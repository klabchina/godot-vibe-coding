# 像素弓箭手角色 - 动画序列帧提示词

## 角色设定

- **角色**: 弓箭手
- **风格**: 现代像素美术 (类似 Dead Cells)
- **视角**: 正上方俯视（上帝视角），从角色正上方正交投影
- **配色**: 深绿色斗篷（带兜帽覆盖头部）、棕色皮革箭袋、木质长弓、深棕色靴子
- **帧数**: 每个动作 5 帧
- **目标帧尺寸**: 128 × 360 像素 (透明背景)

---

## 生成参数

| 参数 | 值 |
|------|-----|
| 生成方式 | image2image（以参考图为布局基准） |
| 参考图 | `seedance-input/rule.png` |
| ratio | 16:9 |
| resolution_type | 2k |
| model_version | 5.0 |
| poll | 120 |

> 输出原始尺寸约 2560×1440（16:9），后续裁切为 5 张独立帧并缩放到 128×360。

### 参考图说明 (`seedance-input/rule.png`)

布局模板图（2560×1440），白底上有 5 个等距黑色矩形区域，定义了角色的精确放置位置：

| 属性 | 值 |
|------|-----|
| 黑色区域尺寸 | 400×650 像素（每个） |
| 顶部留白 | 427 像素 |
| 底部留白 | 363 像素 |
| 左侧留白 / 矩形间距 | 112 像素 |
| 黑色区域行范围 | y=427 ~ y=1076 |

生成时角色必须**完全填满**每个黑色矩形区域，黑色区域外保持纯白背景。

## 核心约束（所有动作必须遵守）

```
⚠️ 重要：以下约束必须在生成时严格执行（基于参考图 seedance-input/rule.png，输出尺寸 2560×1440）

1. 【参考图布局】以 seedance-input/rule.png 为布局基准，图中有 5 个 400×650 像素的黑色矩形区域，
   角色必须完全填满每个黑色区域，黑色区域之外保持纯白色背景。

2. 【统一角色比例】所有 5 帧中角色必须保持完全相同的像素大小，
   角色从头顶到脚底填满黑色区域的 650 像素高度，宽度填满 400 像素宽度。
   三个动作（idle/walk/attack）之间的角色大小也必须完全一致。

3. 【脚底锚点】所有 5 帧角色的脚底必须处于同一水平高度，
   对齐黑色区域的底边（距图片底边 363 像素处），不得上下偏移。

4. 【头顶锚点】角色头顶对齐黑色区域的顶边（距图片顶边 427 像素处），
   所有帧保持一致。

5. 【帧布局】5 帧按参考图中 5 个黑色区域的位置从左到右排列，
   帧与帧之间无任何分隔线、边框、标注或辅助线，
   背景保持纯净纯白，禁止在画面中任何位置出现横线、竖线、分割线、
   网格线、标尺、刻度或任何形式的辅助标记。

6. 【角色位置固定】★ 核心约束 ★ 所有 5 帧中角色必须严格固定在各自黑色区域的
   正中央位置（X轴Y轴均固定），仅允许动作姿态变化，
   绝对禁止角色位置在帧与帧之间出现任何偏移、漂移或位置不一致的问题。

7. 【禁止辅助元素】★ 画面中不得出现任何非角色内容的辅助性视觉元素 ★
   包括但不限于：参考线、对齐线、网格、标注文字、箭头、尺寸标记等。
   整张图片只包含纯白背景和 5 个角色帧，不含任何额外的图形元素。
```

## 后处理：裁切脚本

生成的 sprite sheet（约 2560×1440）需裁切为 5 张独立帧，并缩放到 128×360。透明背景 PNG。

> 注意：使用 image2image + rule.png 参考图后，角色位于 5 个 400×650 的区域内，
> 裁切脚本会自动按内容边界框裁剪并对齐脚底锚点。

```bash
python3 scripts/split_sprite_sheet.py <sprite_sheet.png> <action_name>
```

裁切逻辑：
1. 将 sprite sheet 等分为 5 个区域（每区域约 512×1440）
2. 每帧去除白色背景（阈值 > 240 视为白色，转为透明）
3. 裁剪到角色内容边界框 + 5px 内边距
4. **验证脚底对齐**：若各帧裁剪后内容高度差异 > 5px，则以脚底为锚点统一缩放至相同高度，再等比缩放至 360px 高
5. 保持宽高比缩放至 360px 高（宽按比例缩小至 ≤ 128px），使用最近邻插值（NEAREST）保留像素锐度
6. 居中放置在 128 × 360 透明画布上（宽128 × 高360）
