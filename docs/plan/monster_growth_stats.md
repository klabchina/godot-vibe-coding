## 2. 波次成长系数

怪物属性随波次递增，越后期越强。成长应用于 **HP** 和 **伤害**，速度不变（避免后期怪物追不上变成无法闪避）。

### 2.1 成长公式

```
实际 HP  = 基础 HP  × (1 + 0.10 × (当前波次 - 首次出场波次))
实际伤害 = 基础伤害 × (1 + 0.08 × (当前波次 - 首次出场波次))
```

- 每波 HP 成长 **+10%**（基于基础值加法叠加）
- 每波伤害成长 **+8%**（基于基础值加法叠加）
- 首次出场的波次成长系数 = 0（即该波使用基础值）
- 速度、体型不随波次变化

## 3. 怪物死亡掉落经验

### 3.1 基础掉落 XP

| 怪物类型 | 基础 XP | 设计依据 |
|----------|---------|----------|
| **Slime** | 5 | 最弱小怪，量大补偿 |
| **Skeleton** | 8 | 中等难度，XP 适中 |
| **Orc** | 15 | 高危目标，高回报 |
| **Elite** | 30 | 精英怪，击杀奖励丰厚 |
| **Boss** | 100 | 最终目标，大量经验（但此时已满级，计入分数） |

### 3.2 经验成长

掉落 XP 也随波次成长，与 HP 成长系数一致（怪物越强，经验越多）：

```
实际 XP = 基础 XP × (1 + 0.10 × (当前波次 - 首次出场波次))
```

取整规则：**向上取整**（ceil），保证成长有效。

## 4. 怪物掉落道具

除经验球外，怪物死亡有 **5% 概率**额外掉落临时道具（与经验球独立判定）：

| 道具 | 掉落概率 | 效果 |
|------|----------|------|
| 生命药水 | 2.0% | 恢复 25% 最大 HP |
| 狂暴 | 1.5% | 射速 ×2，持续 5 秒 |
| 无敌 | 0.5% | 免疫伤害，持续 3 秒 |
| 全屏炸弹 | 1.0% | 对全屏怪物造成 50 伤害 |

> 道具掉落不随波次成长，概率和效果固定。Boss 死亡 **必掉** 1 个随机道具。
