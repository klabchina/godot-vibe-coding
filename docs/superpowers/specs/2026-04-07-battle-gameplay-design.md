# 战斗玩法设计：弓箭幸存者

> 双人合作弓箭塔防 — 自动射击 + 走位闪避 + 局内升级

## 概述

玩法灵感来源于「吸血鬼幸存者」，核心体验定位为**轻松休闲、爆爽清屏**。两名玩家在封闭竞技场中自由移动，弓箭自动射击最近敌人，玩家只需专注走位闪避。每波结束后从 3 选 1 升级中不断变强，形成"越打越爽"的正反馈曲线。

**单局时长**：4-6 分钟（8 波制，最多 8 次升级）

---

## 1. 核心战斗循环

### 1.1 基本玩法

两名玩家在一个封闭矩形竞技场中自由移动。弓箭**自动射向最近的怪物**，玩家只需用**摇杆/WASD 走位闪避**怪物。怪物从场地四周刷新，朝玩家移动，接触即造成伤害。

### 1.2 单局流程

```
开始 → Wave 1（简单）→ 升级选择 → Wave 2 → 升级选择 → ... → Wave 8（Boss波）→ 结算
```

- 总共 **8 波**，单局约 4-6 分钟
- 每波持续 20-35 秒
- 波次间歇 **5 秒**，期间弹出升级选择面板
- 两名玩家各自独立选择升级（不需要等对方选完）

### 1.3 操作设计

| 操作 | 方式 | 说明 |
|------|------|------|
| 移动 | WASD / 左摇杆 | 八方向自由移动 |
| 射击 | 自动 | 自动锁定最近敌人，按固定射速发射 |
| 升级选择 | 点击/触屏 | 波次间歇时 3 选 1 |

无需手动瞄准/蓄力/释放——所有射击行为自动进行，玩家全部精力集中在走位。

### 1.4 胜负条件

- **胜利**：清完所有波次（包括最终 Boss 波）
- **失败**：两名玩家的 HP 都降为 0（任一玩家存活则继续）
- 玩家有独立血量，被怪物碰到扣血
- 死亡后 **10 秒自动复活**（每局限 1 次复活）

## 2. 自动射击系统

### 2.1 基础参数

| 参数 | 初始值 | 说明 |
|------|--------|------|
| 箭矢数量 | 1 | 每次发射的箭矢数（可升级） |
| 射速 | 每 0.8 秒 1 发 | 攻击间隔（可升级） |
| 箭矢速度 | 400 px/s | 飞行速度 |
| 基础伤害 | 10 | 每支箭的伤害 |
| 射程 | 无限 | 箭矢飞出屏幕边界销毁 |

### 2.2 自动瞄准规则

1. 搜索屏幕内所有存活怪物
2. 选择**距离玩家最近**的怪物作为目标
3. 箭矢朝目标当前位置方向发射（不做预判）
4. 如果无目标，不发射

### 2.3 箭矢行为

- 箭矢发射后沿直线飞行，**不追踪**
- 默认命中一个怪物后消失
- 可通过升级获得**穿透**（穿过怪物继续飞行，最多穿 N 个）

### 2.4 多人视角

- 两个玩家各自独立执行自动射击逻辑
- 各自锁定各自最近的怪物（自然形成分工——两人分散时各守一侧）

## 3. 局内升级系统

### 3.1 升级流程

1. 每波结束后，每个玩家弹出 **3 选 1** 升级面板
2. 从升级池中随机抽取 3 个升级项展示
3. 选择后立即生效，面板关闭
4. 有 **5 秒倒计时**——超时未选则自动选第一个
5. 两人独立选择，不需要等对方

### 3.2 升级池

#### 攻击类（出现概率 50%）

| 升级名 | 效果 | 可叠加 |
|--------|------|--------|
| 多重射击 | 同时发射箭矢 +1（扇形展开） | 是，最多 8 |
| 射速提升 | 攻击间隔 -15% | 是，最多 5 次 |
| 伤害提升 | 基础伤害 +30% | 是，最多 5 次 |
| 穿透箭 | 箭矢可穿透 +1 个目标 | 是，最多 3 |
| 弹射箭 | 箭矢命中后弹射到附近敌人 | 首次获得 |
| 爆炸箭 | 命中时产生小范围 AOE 伤害 | 首次获得 |

#### 防御类（出现概率 30%）

| 升级名 | 效果 | 可叠加 |
|--------|------|--------|
| 生命提升 | 最大 HP +20% 并恢复等量 | 是 |
| 移速提升 | 移动速度 +15% | 是，最多 3 次 |
| 护盾 | 每 15 秒生成一个可吸收 1 次伤害的护盾 | 首次获得 |
| 生命恢复 | 每秒恢复 1% 最大 HP | 首次获得 |

#### 特殊类（出现概率 20%）

| 升级名 | 效果 | 可叠加 |
|--------|------|--------|
| 磁铁 | 经验球/道具自动吸附范围 +50% | 是 |
| 冰冻箭 | 命中怪物减速 30%，持续 2 秒 | 首次获得 |
| 火焰箭 | 命中附带灼烧 DoT（每秒 3 伤害，3 秒） | 首次获得 |
| 旋转护卫 | 1 个箭矢围绕玩家旋转，碰到怪物造成伤害 | 是，+1 个护卫 |

### 3.3 设计原则

- 每次选择都能感受到明显变强——不做太微小的数值提升
- 升级间有组合效应——如「多重射击 + 穿透 + 爆炸箭」形成满屏爆炸的终极形态
- 8 波 = 8 次升级，足以让角色从"一个小弓箭手"成长为"满屏弹幕机器"

## 4. 波次与怪物设计

### 4.1 波次节奏（8 波制）

| 波次 | 怪物组成 | 总数量 | 持续时间 | 难度感受 |
|------|----------|--------|----------|----------|
| 1 | Slime ×10 | 10 | ~20s | 热身，熟悉操作 |
| 2 | Slime ×8 + Skeleton ×4 | 12 | ~22s | 引入闪避型敌人 |
| 3 | Slime ×5 + Skeleton ×8 + Orc ×2 | 15 | ~25s | 压力上升，Orc 冲锋 |
| 4 | Skeleton ×8 + Orc ×4 + Elite ×1 | 13 | ~25s | 精英怪首次出现 |
| 5 | Skeleton ×6 + Orc ×6 + Elite ×3 | 15 | ~28s | 精英怪增多 |
| 6 | Orc ×8 + Elite ×5 + Skeleton ×5 | 18 | ~30s | 怪物海洋，考验走位 |
| 7 | Orc ×10 + Elite ×8 + Skeleton ×8 | 26 | ~32s | 高压混战 |
| 8 | **Boss** ×1 + Orc ×5 + Elite ×3 + Slime ×10 | 19 | ~35s | 最终决战 |

### 4.2 怪物属性

| 类型 | HP | 速度 | 伤害 | 特性 |
|------|-----|------|------|------|
| **Slime** | 20 | 60 | 5 | 基础，直线追踪 |
| **Skeleton** | 40 | 90 | 8 | 间歇性变速闪避（短暂加速侧移） |
| **Orc** | 80 | 50→150 | 15 | 接近后冲锋（加速冲向玩家，冲锋后短暂眩晕） |
| **Elite** | 150 | 70 | 12 | 普通怪的强化版，体型更大，发光标识 |
| **Boss** | 500 | 40 | 25 | 三阶段：移动追击→召唤小怪→狂暴加速 |

### 4.3 怪物刷新方式

- 从场地**四个边缘随机位置**生成
- 每波怪物在波次开始后**逐个快速刷出**（0.3-0.5 秒间隔），不是同时出现
- 生成位置尽量分散，避免全部从一侧涌入

### 4.4 Boss 设计（第 8 波）

Boss 有三个阶段，每减少 1/3 血量切换：

1. **追击阶段**：缓慢追踪玩家，正常受伤
2. **召唤阶段**：停下来，每 3 秒召唤 3 个 Slime，持续 10 秒
3. **狂暴阶段**：速度翻倍，体型缩小（更难打），不再召唤

## 5. 经验与掉落系统

### 5.1 经验球

- 怪物死亡后掉落**经验球**（小绿色光点）
- 玩家接近经验球时自动拾取（拾取半径 50px，可通过「磁铁」升级增大）
- 经验球**不直接用于升级**——升级节点固定在波次结束时
- 经验球的作用：积累分数，用于结算评分

### 5.2 道具掉落

怪物有小概率（5%）掉落临时道具，拾取后立即生效：

| 道具 | 效果 | 持续时间 |
|------|------|----------|
| 生命药水 | 恢复 25% 最大 HP | 即时 |
| 狂暴 | 射速翻倍 | 5 秒 |
| 无敌 | 免疫所有伤害 | 3 秒 |
| 全屏炸弹 | 对所有怪物造成 50 点伤害 | 即时 |

- 道具掉落概率固定，不因波次变化
- 同一时间只能有一个增益型道具生效（新的覆盖旧的）
- 药水和炸弹是即时效果，不受限制

### 5.3 结算评分

| 评分维度 | 计算方式 |
|----------|----------|
| 波次进度 | 清完所有波次得满分 |
| 击杀数 | 个人击杀数量 |
| 伤害输出 | 个人总伤害 |
| 存活能力 | 剩余 HP 比例 |
| 经验球收集 | 收集总量 |

评分汇总为 **S/A/B/C** 四个等级。

## 6. 对现有架构的影响

### 6.1 新增 Component

| Component | 说明 |
|-----------|------|
| `AutoAimComponent` | 自动瞄准配置（射程、锁定目标 ID、射击间隔计时器） |
| `UpgradeComponent` | 玩家当前所有升级状态（多重射击等级、射速乘数等） |
| `PickupComponent` | 可拾取物（经验球/道具），含类型和值 |
| `EffectComponent` | 箭矢附加效果（穿透、弹射、爆炸、冰冻、火焰等） |
| `BuffComponent` | 玩家身上的临时增益（狂暴、无敌等，含剩余时间） |
| `OrbitComponent` | 旋转护卫箭配置（数量、旋转速度、伤害） |
| `BossPhaseComponent` | Boss 阶段管理（当前阶段、阶段切换血量阈值） |

### 6.2 新增 System

| System | 执行位置 | 说明 |
|--------|----------|------|
| `AutoAimSystem` | 替代原 InputSystem 的射击部分 | 自动搜索最近目标、朝目标发射箭矢 |
| `PickupSystem` | 在 CollisionSystem 后 | 检测玩家与经验球/道具的碰撞，触发拾取 |
| `BuffSystem` | 在 DamageSystem 前 | 更新 buff 剩余时间、应用 buff 效果 |
| `EffectSystem` | 在 DamageSystem 后 | 处理箭矢附加效果（AOE 爆炸、弹射生成新箭矢等） |
| `OrbitSystem` | 在 MovementSystem 后 | 更新旋转护卫箭位置 |
| `BossAISystem` | 在 WaveSpawnSystem 后 | Boss 阶段切换逻辑、召唤小怪 |

### 6.3 修改 System

| System | 变更 |
|--------|------|
| `InputSystem` | 只处理移动输入，移除射击逻辑 |
| `BowSystem` | 移除蓄力逻辑，改为根据 AutoAimComponent 自动发射 |
| `CollisionSystem` | 增加穿透判定、经验球/道具碰撞检测 |
| `DamageSystem` | 支持 AOE 伤害、DoT 伤害 |

### 6.4 更新后的 System 执行顺序

```
 1. InputSystem          — 移动输入
 2. NetworkRecvSystem    — 网络接收
 3. WaveSpawnSystem      — 波次生成
 4. BossAISystem         — Boss AI
 5. AutoAimSystem        — 自动瞄准射击
 6. MovementSystem       — 移动
 7. OrbitSystem          — 旋转护卫
 8. CollisionSystem      — 碰撞检测（箭矢 vs 怪物 + 玩家 vs 拾取物）
 9. PickupSystem         — 拾取处理
10. DamageSystem         — 伤害结算
11. EffectSystem         — 附加效果处理
12. BuffSystem           — Buff 更新
13. DeathSystem          — 死亡清理 + 生成掉落物
14. NetworkSendSystem    — 网络发送
15. RenderSystem         — 渲染
```

### 6.5 网络协议变更

**新增消息：**

| 消息 | 方向 | 说明 |
|------|------|------|
| `UpgradeOptions` | Server → Client | 波次结束时下发 3 个升级选项 |
| `UpgradeChoice` | Client → Server | 玩家选择的升级项 |
| `PickupSpawn` | Server → Client | 掉落物生成通知 |
| `PickupCollect` | Server → Client | 掉落物拾取通知 |
| `BuffApply` | Server → Client | Buff 应用/移除通知 |
| `BossPhaseChange` | Server → Client | Boss 阶段切换通知 |

**修改消息：**

- `PlayerInput`：移除 `aim_angle`、`shoot`、`charge_power` 字段，只保留 `move_dir`
- `PlayerState`：新增 `buffs`、`upgrades` 字段
- `ArrowState`：新增 `effects` 字段（穿透/弹射/爆炸等标记）

### 6.6 UI 新增

| UI | 说明 |
|----|------|
| `UpgradePanel` | 波次间歇弹出的 3 选 1 升级面板，含倒计时 |
| `BuffBar` | 战斗 HUD 上显示当前生效的临时 buff 图标和剩余时间 |
| `UpgradeBar` | 战斗 HUD 上显示已获得的永久升级图标列表 |
| `DamageNumber` | 伤害飘字 |
